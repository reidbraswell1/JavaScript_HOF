JS: Higher-Order Functions

Open the Browser Console to view your work (Right-Click => Inspect or fn+F12)

Function number controls the type of function returned. For example if you enter 25 for the function number then you will get something like this function returned. 

plus25 = plus(2nd value);

In the parenthesis the 2nd value will be one you enter "Enter Number to add to:" input element. In the example the 2nd number would be added to 25 assuming you entered 25 for the first value.
